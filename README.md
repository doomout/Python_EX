            if, elif, else를 사용한다.  
```py
#if today.weekday() in [5,6]: 으로 줄일 수도 있다.  
if today.weekday() == 6 or today.weekday() == 5:  
      print("주말이니 회사에 가지 않아!!")  
      print("온종일 신나게 놀즈아~!")  
elif today.weekday() == 4:   
      print("예아!! 오늘은 불금이다!!!")  
else:   
      print("오늘은 회사 가는 날....")  
```
